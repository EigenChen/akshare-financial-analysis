# 快速开始指南

## 第一步：安装依赖

打开终端（命令行），在项目目录下运行：

```bash
pip install -r requirements.txt
```

如果安装较慢，可以使用国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 第二步：验证安装

运行第一个学习脚本，验证 AKShare 是否正确安装：

```bash
python 01_基础学习_验证安装.py
```

如果看到 "✓ AKShare 安装成功，可以正常使用！"，说明安装成功。

## 第三步：按顺序学习

按照以下顺序运行学习脚本，每个脚本都有详细注释：

1. **01_基础学习_验证安装.py** - 验证安装
2. **02_基础学习_获取股票列表.py** - 学习获取股票列表
3. **03_基础学习_获取财务数据.py** - 学习获取财务数据
4. **04_基础学习_获取年报数据.py** - 专门学习获取年报数据
5. **05_实战_年报数据收集.py** - 实战批量收集数据

## 学习建议

1. **不要急于求成**：每个脚本都要理解清楚再继续
2. **动手实践**：修改代码中的股票代码，尝试获取不同股票的数据
3. **查看数据结构**：运行脚本后，仔细查看返回的数据结构
4. **阅读注释**：每个脚本都有详细注释，帮助理解
5. **参考文档**：遇到问题查看 `学习指南.md`

## 常见问题

### Q: 运行脚本时提示 "ModuleNotFoundError: No module named 'akshare'"
A: 说明 AKShare 没有安装成功，重新运行 `pip install -r requirements.txt`

### Q: 获取数据时提示网络错误
A: 检查网络连接，某些接口可能需要访问特定网站

### Q: 数据获取失败
A: 
- 检查股票代码是否正确
- 某些股票可能没有数据
- 查看错误信息，可能需要调整参数格式

### Q: 如何知道某个接口的详细用法？
A: 
- 查看 AKShare 官方文档：https://akshare.akfamily.xyz/
- 在 Python 中使用 `help()` 函数：`help(ak.stock_info_a_code_name)`
